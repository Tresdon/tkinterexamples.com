Events

Events in tkinter are things that happen inside your application such as mouse clicks & movements, keyboard entries, window resizes, widget introduction, the list goes on. Generally there are two types of events, those which are built into a widget and those which are explicitly bound to a widget.

Implicit Events

Some widgets have events built into them such as the Button widget which takes a command= argument. This implicitly binds the event for clicking the button to whatever procedure the command argument specifies. Some other widgets with built in event bindings (via command= on state change) are:

  • Checkbutton
  • Entry
  • Radiobutton
  • Scale
  • Scrolbar
  • Spinbox
